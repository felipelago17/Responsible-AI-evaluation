# Robustness Evaluation

This benchmark evaluates the stability and reliability of LLM outputs when inputs are perturbed, rephrased, or subjected to adversarial manipulation.

## Overview

| Property | Value |
|---|---|
| Primary Metric | Accuracy Under Attack |
| Dataset | AdvGLUE + PromptBench + Custom |
| Dataset Version | Pinned (see `benchmarks/robustness/config.yaml`) |
| Evaluation Type | Classification + Generation |

## Robustness Dimensions

### Adversarial Text Perturbations

Measures accuracy when inputs contain character-level (typos, character swaps), word-level (synonym substitutions, word deletions), and sentence-level (paraphrases) perturbations.

### Prompt Sensitivity

Measures consistency of outputs across semantically equivalent phrasings of the same prompt. A robust model should produce consistent answers regardless of how a question is phrased.

### Prompt Injection Resistance

Measures the model's resistance to prompt injection attacks where malicious instructions are embedded in user inputs in an attempt to override system behaviour.

### Out-of-Distribution Generalisation

Measures performance degradation when inputs come from domains or distributions not well-represented in the model's training data.

### Instruction Following Under Noise

Measures whether the model correctly follows instructions when they contain minor grammatical errors, ambiguities, or distractors.

## Evaluation Protocol

### AdvGLUE

The model is evaluated on adversarially modified versions of standard NLU tasks (SST-2, QQP, MNLI, QNLI, RTE). Perturbations include word substitutions generated by human annotators and automated adversarial attack algorithms.

### PromptBench

The model is evaluated across 13 adversarial prompt attack types including TextFooler, BERTAttack, and character-level attacks, applied to prompts for 8 NLP tasks.

### Prompt Injection

A custom test suite of prompt injection scenarios covering direct injection, indirect injection via retrieved content, and multi-turn injection attacks.

## Metrics

| Metric | Description | Target |
|---|---|---|
| Accuracy Under Attack (primary) | Accuracy on adversarially perturbed inputs | Higher is better |
| Attack Success Rate | Fraction of attacks that change model output | Lower is better |
| Consistency Score | Agreement across paraphrased prompt variants | Higher is better |
| Injection Resistance Rate | Fraction of injection attempts successfully deflected | Higher is better |
| Performance Drop | Accuracy reduction vs. clean inputs | Lower is better |
